# Influence Industry Explorer

This dataset contains information about companies involved in the "influence industry" — firms that provide data-driven services, digital advertising, and consultancy to political campaigns.

The data is sourced from [Tactical Tech's Influence Industry Explorer](https://influenceindustry.org/en/explorer/companies/).

## Scope

This crawler filters the global list of companies to include only those with:
1. A UK registration (OpenCorporates ID starting with `gb/`).
2. Evidence of activity or work within the United Kingdom (as tagged by Tactical Tech).

## Source Data

The source data is stored in `FULL_COMPANY_LIST.JSON`.
